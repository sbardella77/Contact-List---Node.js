# Contact-List---Node.js
Simple Contact List App using Node.js without Database



### What do you need

```sh
npm install
```
```sh
npm init
```
```sh
npm install express --save
```
 ```sh
npm install pug
```

```sh
npm install --save multer
```

```sh
npm install --save-dev nodemon
```
